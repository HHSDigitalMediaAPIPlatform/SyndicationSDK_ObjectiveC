getSourceById:success:failure:

Request a specific source object from the Syndication API by source Id

- (void)getSourceById:(NSUInteger)sourceId success:(void ( ^ ) ( SynSourceResults *sourceResults ))success failure:(void ( ^ ) ( SynSourceResults *sourceResults , NSError *error ))failure

Parameters

sourceId

The campaign Id to request information about

success

A block object to be executed when the object request operation finishes successfully. This block has no return value and takes one argument: the created SynSourceResults object that contains the results of the request.

failure

A block object to be executed when the request operation finished unsuccessfully, or that finishes successfully, but encountered an error while parsing the response data. This block has no return value and takes two arguments: the SynSourceResults object that was created and the NSError object describing the network or parsing error that occurred.

getSourcesWithOptions:success:failure:

Request source information from the Syndication API using options

- (void)getSourcesWithOptions:(NSDictionary *)options success:(void ( ^ ) ( SynSourceResults *sourceResults ))success failure:(void ( ^ ) ( SynSourceResults *sourceResults , NSError *error ))failure

Parameters

options

NSDictionary containing various options (see Discussion below)

success

A block object to be executed when the object request operation finishes successfully. This block has no return value and takes one argument: the created SynSourceResults object that contains the results of the request.

failure

A block object to be executed when the request operation finished unsuccessfully, or that finishes successfully, but encountered an error while parsing the response data. This block has no return value and takes two arguments: the SynSourceResults object that was created and the NSError object describing the network or parsing error that occurred.

Discussion

The options parameter takes a NSDictionary with the following valid keys.

IMPORTANT the values MUST be Objective C objects (eg: @“1” instead of 1)

max: The maximum number of records to return offset: The offset of the records set to return for pagination. sort: Which field to sort the records by.
