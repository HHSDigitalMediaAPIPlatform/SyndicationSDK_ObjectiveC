getMediaByCampaignId:options:success:failure:

Get a list of media items for a specific campaign based on options.

- (void)getMediaByCampaignId:(NSUInteger)campaignId options:(NSDictionary *)options success:(void ( ^ ) ( SynMediaResults *mediaResults ))success failure:(void ( ^ ) ( SynMediaResults *mediaResults , NSError *error ))failure

Parameters

campaignId

The campaign Id to return the media items from

options

NSDictionary containing various options (see Discussion below)

success

A block object to be executed when the object request operation finishes successfully. This block has no return value and takes one argument: the created SynMediaResults object that contains the results of the request.

failure

A block object to be executed when the request operation finished unsuccessfully, or that finishes successfully, but encountered an error while parsing the response data. This block has no return value and takes two arguments: the SynMediaResults object that was created and the NSError object describing the network or parsing error that occurred.

Discussion

The options parameter takes a NSDictionary with the following valid keys.

IMPORTANT the values MUST be Objective C objects (eg: @“1” instead of 1)

 max:                            The maximum number of records to return
 offset:                         The offset of the records set to return for pagination.
 sort:                           Which field to sort the records by.
